# 📝 Structured Notes Management API

A modular REST API built with Node.js and Express for creating, managing, and filtering structured notes with JWT-based authentication.

---

## ✨ Overview

This project was built to explore backend fundamentals and scalable API design:

- JWT-based authentication and protected routes
- Tag-based dynamic filtering
- Keyword search with pagination
- Layered backend architecture
- Centralized error handling and input validation

---

## 🧠 Architecture

The API follows a clean layered structure with clear separation of concerns:

- **Routes** — Define API endpoints and map to controllers
- **Controllers** — Handle request/response logic
- **Services** — Contain core business logic
- **Middleware** — Validation, authentication, and error handling

This structure keeps each layer focused and independently maintainable.

---

## ⚙️ Core Features

- **JWT Authentication** — Secure login and protected resource access
- **Tag-Based Filtering** — Filter notes dynamically using one or multiple tags
- **Keyword Search** — Full-text search across note content
- **Date Range Queries** — Filter notes by creation date range
- **Pagination** — Efficient paginated result handling
- **Input Validation** — Structured validation middleware on all endpoints
- **Global Error Handling** — Centralized error handler for consistent API responses

---

## 🛠 Core Elements

- Node.js
- Express.js
- MongoDB
- JWT
- REST Architecture
- Layered MVC-style structure
